Weather Website
Overview
The Weather Website provides real-time weather updates and forecasts for any location worldwide. It uses the OpenWeatherMap API to fetch and display accurate weather information, ensuring users stay informed about current conditions and future weather patterns.

Features
Real-Time Weather Updates: Current weather information for any city.
7-Day Forecast: Detailed weather forecasts for the upcoming week.
Responsive Design: Accessible on both desktop and mobile devices.
Technologies Used
Frontend: HTML, CSS, JavaScript
API: OpenWeatherMap API
